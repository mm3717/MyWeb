Info
Welcome to my website!
I've styled the user interface after the unique aesthetic of Windows 98. One of my hobbies is experimenting with CSS, and I thought it would be interesting to emulate an operating system with draggable windows in the browser. Be sure to check out the projects "app" in the taskbar. This will take you to some of the projects I’ve saved from my time at university (still working on integrating more!). I'm quite proud of the projects I've completed individually, as well as the ones made in collaboration with colleagues. I hope this site serves as an effective portfolio to demonstrate some of the technical skills listed on my resume. Where I couldn’t share GitHub links due to confidentiality, I’ve included detailed project documentation instead. I hope you enjoy browsing through these artifacts and interacting with the website (remember, windows can be resized)!
